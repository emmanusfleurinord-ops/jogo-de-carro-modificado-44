## 🎮 Como jogar
| Tecla | Ação |
|-------|------|
| `←` `→` | Mudar de faixa |
| `↑` | Acelerar |
| `↓` | Frear |
Desvie dos carros inimigos. Você tem **3 vidas** — a cada colisão perde uma. A velocidade aumenta com o tempo!
